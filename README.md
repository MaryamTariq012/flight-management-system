# Flight Management System

A comprehensive desktop GUI application integrated with a relational database system to manage airline operations, passenger bookings, and flight scheduling.

## Features
- Dynamic User Interface designed using C# Windows Forms (.cs layout components).
- Architected optimized relational database schemas for tracking airline matrices and customer records.
- Implemented multi-table storage constraints, primary/foreign keys, and normalized structural joins.
- High-performance SQL queries for automated data extraction and database record updates.

## Tech Stack
- Frontend/GUI: C# (.NET Windows Forms)
- Database Engine: Microsoft SQL Server / MySQL
- IDE: Visual Studio 2022
- Language: C#, SQL
